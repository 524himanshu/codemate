# CodeMate Frontend

This directory contains the Next.js web application for CodeMate.

For the complete project features list, architecture overview, and full environment setup instructions, please refer to the **[Root README](../README.md)**.

## Quick CLI Commands

To start the frontend locally:

```bash
# 1. Install dependencies
npm install

# 2. Run the development server
npm run dev
```

The app will be active at [http://localhost:3000](http://localhost:3000).
